Open Data

The Star Wars API.

This web application makes use of Open Data to provide an engaging and meaningful service. What exactly is open data? The Open Data Handbook describes open data in the following way: Open data is data that can be freely used, re-used and redistributed by anyone - subject only at most, to the requirement to attribute and sharealike.

To summarize the most important aspects:

  • Availability and Access: the data must be available as a whole and at no more than a reasonable reproduction cost, preferably by downloading over the internet. The data must also be available in a convenient and modifiable form.
  • Re-use and Redistribution: the data must be provided under terms that permit re-use and redistribution including the intermixing with other datasets.
  • Universal Participation: everyone must be able to use, re-use and redistribute - there should be no discrimination against fields of endeavour or against persons or groups. For example, ‘non-commercial’ restrictions that would prevent ‘commercial’ use, or restrictions of use for certain purposes (e.g. only in education), are not allowed.

Specifically this application makes use of the Star Wars API (SWAPI). The Star Wars API is the world's first quantified and programmatically-formatted set of Star Wars data. It is comprised of data of all the People, Films, Species, Starships, Vehicles and Planets from Star Wars. The data is formatted in JSON and exposed to users in a REST implementation that allows developers to programmatically collect and measure the data. You can find more about SWAPI at the official website found here SWAPI

An excellent example of an open data resource is ProgrammableWeb. ProgrammableWeb is the world's leading source of news and information about Internet-based application programming interfaces (APIs). It provides a directory that you can use when discovering and searching for APIs to use in Web and mobile applications. For more information, you can check out the official site here ProgrammableWeb
